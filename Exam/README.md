# E-commerce Exam Project (Node.js + Express + MongoDB + EJS)

## Setup

1. Copy `.env.example` to `.env` and fill MONGO_URL and JWT_SECRET.
2. Install dependencies:
   ```
   npm install
   ```
3. Run dev server:
   ```
   npm run dev
   ```
4. Open http://localhost:3000

## Features
- Register / Login (cookies + JWT)
- Roles: admin / user
- Create / Edit / Delete Products
- Categories management (admin)
- EJS views (simple UI)
